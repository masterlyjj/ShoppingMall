## 6. 경영자 / 관리자 기능
/manager페이지로 이동해서 관리자 / 경영자 계정으로 별도 로그인
관리자 경영자는 SQL에서 별도로 계정을 생성한다.
아이템 등록 및 카테고리 생성등 회원 관리기능도 가지고있다.
